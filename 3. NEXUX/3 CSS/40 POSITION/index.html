Position Changing Types and Featured

  • Relative - holds original position - position change from its original position
  • Absolute - Free original position - positon change from Nearest Parent(which have position:relative)
  • Fixed - Free original position - Position change from the vw/vh/px (SCREEN)
  • Sticky - holds original position - Position change "top" ONLY, WITHIN its """Parent"""
