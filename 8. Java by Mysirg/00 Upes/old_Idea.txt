PART 2: SEMESTER-WISE MASTER CHECKLIST
======================================================================

SEMESTER 1 (FOUNDATION)
Subjects: Linux Lab, C, C Lab, Problem Solving, Math-I, Physics, EVS-I, Living Conversation.

Master these:
1. Linux terminal: files, permissions, grep/find, shell scripts, process commands.
2. C basics to advanced: pointers, arrays, strings, functions, structures, files, memory.
3. C lab problem set: loops, matrix ops, recursion, pointer tasks, file handling.
4. Problem-solving frameworks: Polya method, logic puzzles, counting basics.
5. Math-I: calculus, vector calculus, ODE basics.

Output by end of Sem 1:
1. 2 C mini projects.
2. 1 shell scripting toolkit.
3. 100 coding problems (easy level).


SEMESTER 2 (CORE BUILD)
Subjects: COA, DSA, DSA Lab, Python, Python Lab, Digital Electronics, Math-II, EVS-II, Critical Thinking.

Master these:
1. COA: number systems, instruction cycle, memory hierarchy, pipelining.
2. DSA: arrays, linked list, stack, queue, trees, graphs, hashing, sorting/searching.
3. Python: OOP, files, exceptions, libraries, numpy/pandas/matplotlib basics.
4. Digital Electronics: combinational and sequential circuits, logic minimization.
5. Math-II: numerical methods, optimization, complex analysis, PDE basics.

Output by end of Sem 2:
1. 2 DSA implementations repo (C/Python).
2. 1 Python data project.
3. 200 cumulative coding problems.


SEMESTER 3 (COMPUTING CORE)
Subjects: AIML, AIML Lab, DBMS, DBMS Lab, DAA, DAA Lab, Discrete Math, OS, Exploratory-1, Design Thinking.

Master these:
1. DBMS: ER, normalization, SQL, transactions, indexing, NoSQL intro.
2. DAA: divide-conquer, greedy, DP, backtracking, NP completeness basics.
3. OS: processes, scheduling, synchronization, deadlocks, memory/file systems.
4. Discrete Math: logic, combinatorics, graphs, number theory.
5. AIML: supervised vs unsupervised, preprocessing, model evaluation.

Output by end of Sem 3:
1. 1 SQL + Java/Python DB app.
2. 1 DAA algorithm notebook.
3. 1 OS simulation mini project.


SEMESTER 4 (SYSTEM + DESIGN)
Subjects: CN, CN Lab, OOP, OOP Lab, Software Engineering, Linear Algebra, PE-1, Exploratory-2.

Master these:
1. CN: layers, routing, congestion, TCP/UDP, wireshark basics.
2. OOP Java: classes, inheritance, interfaces, exceptions, collections, JDBC.
3. SE: SDLC, requirements, testing, metrics, risk, quality models.
4. Linear Algebra: vector spaces, eigenvalues, SVD.
5. PE-1 according to track.

Output by end of Sem 4:
1. 1 full Java OOP + DBMS project.
2. 1 network lab diagnostics repository.


SEMESTER 5 (SPECIALIZATION DEPTH)
Subjects: Cryptography, FLAT, OOAD, Research Methodology, Probability/Entropy/MC, PE-2, Exploratory-3.

Master these:
1. Security primitives: symmetric/asymmetric crypto, hashes, protocols.
2. FLAT for compiler and theory base.
3. OOAD and UML models.
4. Research writing: papers, literature review, proposal.
5. Probability/statistics for data and ML depth.

Output by end of Sem 5:
1. 1 security-focused implementation.
2. 1 research-style report.


SEMESTER 6 (BUILD + APPLY)
Subjects: Compiler Design, Statistics and Data Analysis, Leadership, PE-3, Minor Project, Exploratory-4.

Master these:
1. Compiler phases, parsing, symbol table, optimization basics.
2. Stats for modeling and analysis.
3. Team execution and project management.
4. Minor project as production-grade work.

Output by end of Sem 6:
1. 1 deployable minor project.
2. Full documentation + demo video + tests.


SEMESTER 7
Subjects: PE-4, PE-5, Capstone Phase-1, Exploratory-5, Summer Internship.

Master these:
1. Advanced elective depth.
2. Capstone architecture and implementation strategy.
3. Internship problem solving and team communication.


SEMESTER 8
Subjects: IT Ethical Practices, Capstone Phase-2.

Master these:
1. Responsible AI, privacy, ethics, professional practice.
2. Capstone completion, deployment, publication, presentation.
